Dans ce projet personnelle, j'ai réussi a réaliser une application E-commerce complète, en utilisant Django pour le Backend et React pour le Frontend, Avec les fonctionnalités suivantes:

  1. Panier d'achat complet.
  2. Commentaires et évaluations des produits.
  3. Fonction de recherche de produits.
  4. Profil de l'utilisateur avec commandes.
  5. Gestion des produits par l'administrateur.
  6. Gestion des utilisateurs (Admin).
  7. Processus de paiement (Livraison, mode de paiement, etc.).
  8. Intégration de PayPal / carte de crédit.

L'application a été déployé en utilisant Heroku et Git.

Technologies:
  • - Python
  • - Django + Django REST Framework
  • - React JS
  • - BootStrap
  • - Postgres
  • - Cloudinary
  • - Heroku
